NSArray(FunkyUtilities)

@interface NSArray <__covariant ObjectType>
(FunkyUtilities) @end

This extension provides you with simple and easy to use functional and general utilities for NSArray. If you need to prefix the extension methods in this category, you should import NSArray+FunkyPrefixedUtilities.h, where every utility method is prefixed with the funky_ keyword for compatiblitiy reasons.
